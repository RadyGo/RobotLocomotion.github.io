symbolic_expression_cell.h File Reference

Detailed Description

Provides implementation-details of symbolic expressions.

It is strongly discouraged to include and use this header file outside of drake/common/symbolic_* files. To include this file, you need to define DRAKE_COMMON_SYMBOLIC_DETAIL_HEADER before. Without it, you have compile-time errors.
