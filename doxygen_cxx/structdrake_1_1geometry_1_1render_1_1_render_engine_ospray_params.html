◆ background_color

std::optional<Eigen::Vector3d> background_color {}

The rgb color for the environment background (each channel in the range [0, 1]).

The default value is the same color as in the equivalent byte-valued rgb triple [204, 229, 255].

◆ default_diffuse

std::optional<Eigen::Vector4d> default_diffuse {}

The (optional) rgba color to apply to the (phong, diffuse) property when none is otherwise specified.

◆ samples_per_pixel

int samples_per_pixel {1}

The number of illumination samples per pixel.

Higher numbers introduce higher quality at increased cost. Only has an effect if mode is OsprayMode::kPathTracer.

◆ use_shadows

bool use_shadows {true}

Whether to turn on shadows when in the OsprayMode::kRayTracer rendering mode.

It is ignored in other modes.
